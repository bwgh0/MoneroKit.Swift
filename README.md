# MoneroKit.Swift

`MoneroKit.Swift` is a Swift package that provides Monero wallet functionality for iOS applications. It wraps the Monero wallet2 C++ library and includes support for light wallet server (LWS) mode.

## Acknowledgments

This project is based on [MoneroKit.Swift](https://github.com/horizontalsystems/MoneroKit.Swift) by Horizontal Systems, originally developed for the Unstoppable Wallet.
